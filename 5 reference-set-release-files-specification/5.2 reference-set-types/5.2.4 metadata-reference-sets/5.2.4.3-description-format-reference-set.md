# Description Format Reference Set

## Purpose

The [900000000000538005 |Description format reference set|](http://snomed.info/id/900000000000538005) specifies the text format and maximum length of each supported description type. This permits additional description types to be specified in future in addition to the three existing description types ( synonym, fully specified name and textual definition).

## Data structure

The [900000000000538005 |Description format reference set|](http://snomed.info/id/900000000000538005) is a C-I ( component- Integer) reference set which is used to specify the length and format of the terms in descriptions of this description type. Its structure is shown in the following table.
